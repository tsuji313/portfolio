# Portfolio - Tsuji Rumiko

## 概要

HTML・CSS・JavaScript で作ったポートフォリオサイトです。
コーディングや更新を中心とした業務を希望して求職活動中です。

## 制作目的

Web サイトの更新業務に必要な「正確なコーディング」「軽微な修正」「デザイン調整」などの実務スキルを整理・発信するために制作しました。

## 使用技術

- **HTML5 / CSS3**
  - セマンティックな構造と可読性を意識
  - レスポンシブ対応（モバイル～ PC）
- **JavaScript / jQuery**
  - ハンバーガーメニューやスムーススクロールなど基本動作
- **Git / GitHub**
  - ブランチ操作、リポジトリ管理、コード更新
- **Design Tools**
  - Photoshop / Illustrator：画像加工・ロゴ制作
  - XD：ワイヤーフレーム、デザインカンプ作成
- **その他**
  - FileZilla（サーバーアップロード）

## 構成

- **About**：自己紹介
- **Skills**：対応可能な技術一覧
- **Experience**：これまでの職務経歴と担当業務
- **Works**：実際の制作物（不動産サイト、LP、架空企業・ショップサイトなど）
- **Strengths**：業務で活きる強みを整理
